-------------- JCasC - Jankins As Code -------------------
-------------------- Instractions ------------------------

1.clone the repo from git:
 -" git clone https://github.com/MayElbaz18/Monithor-Work.git "

2.Run Apt Update And Install Docker.io:
 -" apt update -y && apt install -y docker.io "

3. Make A Dirctory For Volume Jenkins Data And Set Permissions: 
 -" mkdir /var/jenkins_data " - To Set This Dir As A Volume For The Jenkins Conatainer.
 -" chown -R 1000:1000 /var/jenkins_data " - To Set Permissions To The Dirctory.
 -" cd Monithor-Work/Casc " - Change Dirctory.
 -" cp -r . /var/jenkins_data " - Copy The Files To The Volume Dirctory.

4.change dirctory to /Monithor-Work/Casc (The git repo folder)
 after the directory set run the following commands:
 
 -" docker build -t jenkins/monithor:casc . " - To Build the Docker Image (If not Located In Docker Hub)
 
 -" docker run --name jenkins --rm -v /var/jenkins_data:/var/jenkins_home -p 8080:8080 --env JENKINS_ADMIN_ID=admin --env JENKINS_ADMIN_PASSWORD=admin jenkins/monithor:casc " - Running The Container With The Envs That Required.

** In Case The Docker Image Is In The Docker Hub - Run This command: **
